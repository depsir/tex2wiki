# Tex2wiki

This gem converts a tex file to mediawiki and uploads its content to a mediawiki url

## Installation

Add this line to your application's Gemfile:

    gem 'tex2wiki'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install tex2wiki
